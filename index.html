WEB Conference

2022. 5. 20. 14:00 - 16:00

Webサイト制作の勉強会、WEB Conference。 今回はCSSのアニメーションテクニックとデザインを中心に紹介します。 参加料金は無料！お気軽にご参加ください。

About

WEB Conferenceとは、Webサイト制作を学びたい人たちが集まる勉強会です。 最新のWebサイト制作やデザイントレンドを紹介し、それらの知識について参加者でディスカッションします。 一人で勉強を続けるだけでなく、仲間と交流しながら楽しく学んでいきましょう。
